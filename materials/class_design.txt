① Entity（スーパークラス）

属性
float x;
float y;
float hp;
float speed;
boolean alive;

メソッド
update();
display();
takeDamage(float damage);
isDead();

メモ
ゲーム中に存在するキャラクターの共通クラス。
Player・Enemy・Bossの親クラス。

② Player

Entityを継承

属性
Weapon currentWeapon;
ArrayList<Weapon> weapons;

float maxHp;
float moveSpeed;
int score;

int level;
int exp;
int nextLevelExp;

メソッド
move();
attack();
switchWeapon();
heal();
gainExp(int exp);
levelUp();
applyUpgrade(Upgrade upgrade);

メモ
プレイヤー操作やステータス（HP、移動速度など）の管理を担当する。経験値を獲得してレベルアップし、Waveクリア時またはレベルアップ時に選択した強化を適用する。

④ Boss

メソッド

メモ

などをここで表現。

⑧ Upgrade
属性
String name;
String description;

String target;    PLAYER または WEAPON
String type;      DAMAGE, RANGE, HP など
float value;

メソッド
なし

メモ
Waveクリア時またはレベルアップ時に選択できる強化を表すクラス。
targetは強化対象（PLAYERまたはWEAPON）、
typeは強化内容、
valueは強化量を表す。
