# 武器切り替え型アリーナシューティング

## アプリ概要

プレイヤーが3種類の武器（射撃モード）を切り替えながら、次々と出現する敵を倒してハイスコアを目指す2Dシューティングゲームです。

敵にはそれぞれ得意・不得意な攻撃属性が設定されており、状況に応じて武器を切り替えることが攻略の鍵となります。ただ敵を撃つだけではなく、「どの武器を使うか」を考えながら戦う戦略性を重視しています。

ゲームは1画面固定のアリーナ形式で進行し、敵をすべて倒すと次のWaveへ進みます。Waveが進むごとに敵の数や種類が増え、最後にはボスが出現します。

また、敵を倒すと経験値オーブがドロップし、プレイヤーが回収することで経験値を獲得できます。一定量の経験値を獲得するとレベルアップし、複数の強化候補の中から1つを選択できます。

さらに、各Waveをクリアした際にも強化を1つ選択できます。プレイヤーの能力や武器性能を強化することでプレイスタイルが変化し、毎回異なる戦略でゲームを楽しめるローグライク要素を取り入れます。

---

## ゲームのルール

* プレイヤーは1画面内を自由に移動できる。
* 3種類の武器（射撃モード）を切り替えながら敵を攻撃する。
* 武器ごとに攻撃属性が異なり、敵によって有効な武器が異なる。
* 敵を倒すと経験値オーブがドロップする。
* 敵をすべて倒すと次のWaveへ進む。
* Waveクリアやレベルアップごとに提示された強化候補から1つを選択し、プレイヤーまたは武器を強化できる。
* Waveが進むにつれて敵は強くなり、出現数も増加する。
* 最終Waveではボスと戦闘を行う。
* プレイヤーのHPが0になるとゲームオーバーとなる。
* 敵を倒した数やWave到達数に応じてスコアを獲得し、ハイスコアを目指す。

---

## 武器の例

| 武器         | 攻撃属性 | 特徴                     |
| ---------- | ---- | ---------------------- |
| 連射モード      | 貫通   | 連射間隔が短く、敵を貫通して複数体にヒットする |
| 大砲モード    | 打撃   | 発射間隔は長いが一発の威力が高い       |
| 爆発弾モード     | 爆発   | 着弾地点周辺の敵に範囲ダメージを与える    |

---

## 敵の特徴

敵ごとに弱点となる攻撃属性が設定されている。

例：

* 重装兵：打撃に弱い
* 飛行型：貫通に弱い
* 障害物型：爆発に弱い

敵の見た目や装備から弱点を推測できるようなデザインを目指す。

---

## ゲームの流れ

1. タイトル画面
2. ゲーム開始
3. Wave開始
4. 敵を撃破
5. 経験値オーブを回収して経験値を獲得
6. レベルアップ時に強化を1つ選択
7. 敵を全滅させるとWaveクリア
8. Waveクリア時に強化を1つ選択
9. 最終Waveでボス戦
10. ボス撃破でゲームクリア
11. スコア表示

---

## 今後追加を検討している機能

時間に余裕がある場合は、現在の1画面固定アリーナ形式から発展させ、広いマップを自由に探索できるモードの実装を検討している。

このモードでは、プレイヤーを中心にカメラが追従し、広いマップ内に配置された敵やボスを探索・撃破しながら進行する。マップ上には敵の種類や配置に変化を持たせ、探索性や戦略性を向上させることを目指す。

また、武器のランダム強化やランダムイベントなどのローグライク要素を追加し、プレイするたびに異なる攻略が楽しめるゲームへ発展させる予定である。
